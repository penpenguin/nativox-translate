# Nativox Translate

WSLg + Electron 上で、環境内の AI Agent（Codex/Claude Code/GeminiCLI など）を
使ってテキスト翻訳するアプリ。

## 開発 (WSLg)

1. 依存関係を導入

```
npm install
```

2. 開発起動

```
npm run dev
```

## 本番ビルド (WSLg)

1. ビルド

```
npm run build
```

2. プレビュー起動

```
npm run preview
```

## テスト

```
npm run test
```

## Lint/Format

```
npm run lint
npm run format
```
